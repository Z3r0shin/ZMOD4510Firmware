This is the firmware package providing the algorithm implementation and
examples for Nitrogen Dioxide and Ozone computation based on ZMOD4510.

# Top-level folder contents:
  windows:       Precompiled example for MS Windows on EVK platform
  raspberrypi:   Precompiled example for Raspberry Pi
  arduino:       Arduino library
  src:           Example source code
  doc:           Documentation
  lib:           Algorithm libraries for static linking with customer applications
